📖 CTA 数据解读指南
🔹 什么是 COT 持仓？
CFTC 每周发布的持仓报告，披露期货市场各类参与者的多空头寸。 Non-Commercial（非商业）= 投机者、CTA 基金、对冲基金。 他们的仓位方向 ≈ 趋势跟踪策略的"共识方向"。
🔹 净持仓 = 多头 − 空头
正值 = 投机者看多（红色）， 负值 = 投机者看空（绿色）。
绝对值越大 → 仓位越拥挤 → 反转风险越高。
🔹 周变化怎么看？
周变化 > +10K = 大幅加多，看涨信号
周变化 < −10K = 大幅加空，看跌信号
关键：净空头 + 周变化转正 = 空头回补开始，可能是底部信号！
🔹 趋势图怎么读？
多品种同方向 = CTA 系统化资金集体行动（最强信号）
品种分化 = 资金在轮动，没有统一方向
极端仓位 + 方向反转 = 大行情可能即将启动
🔹 信号矩阵怎么用？
价格 vs 20/50MA 位置 = CTA 趋势跟踪信号依据
价格 > 20MA > 50MA = 上升趋势确认
价格 < 20MA < 50MA = 下降趋势确认
多品种同时确认同一趋势 → 最高置信度
🔹 Risk-On vs Risk-Off
Risk-On：股指↑ + 原油↑ + 铜↑ + 黄金↓ → 市场追逐风险
Risk-Off：股指↓ + 黄金↑ + 日元↑ → 资金涌入避风港
